Deploy QashqAI Voice to production:

1. Run `npm run build` and fix any errors
2. Run `npm run test` if tests exist
3. Review changes with `git diff`
4. Commit with descriptive message following convention: `feat:`, `fix:`, `docs:`, `chore:`
5. Push to main: `git push origin main`
6. Verify Vercel auto-deploy at https://qashqaivoice.com
7. Check deployment logs if needed
