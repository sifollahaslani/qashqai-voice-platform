# Tech Stack v1

## Purpose
Outline a realistic early technical stack for QashqAI Voice.

## Current working environment
- laptop-based local development
- VS Code
- Python
- Markdown documentation
- structured project folders

## Recommended early stack
### Development
- Python
- VS Code
- GitHub later
- local terminal / PowerShell

### Documentation
- Markdown files
- PDF export later
- DOCX conversion later if needed

### Data handling
- organized local folders
- spreadsheets for metadata
- consent documents
- structured file naming

### Future technical layer
- Python-based preprocessing
- speech/audio tooling
- ASR experimentation
- evaluation scripts
- dataset preparation logic

## Why this stack
- simple
- low-cost
- understandable
- expandable later
- suitable for early institutional and technical preparation

## Not yet required
- cloud infrastructure
- complex deployment
- app backend production
- advanced automation
- scaling architecture

## Current status
The technical environment is at the setup and structuring phase, which is appropriate for the current stage of the project.
