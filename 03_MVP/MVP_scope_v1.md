# MVP Scope v1

## Purpose
The initial MVP of QashqAI Voice is designed to establish a practical and ethical starting point for endangered language preservation through structured documentation, responsible data handling, and future technical development.

## Core goal
Build a minimal but credible foundation that demonstrates:
- project seriousness
- ethical and cultural responsibility
- institutional readability
- future technical viability

## MVP priorities
- create core institutional documents
- define governance and consent logic
- prepare a small, consent-based language documentation pathway
- define a realistic technical workflow for future speech/language support
- prepare materials for outreach and funding dialogue

## MVP components
### 1. Documentation layer
- one-pager
- short bio
- institutional introduction
- project summary
- outreach drafts

### 2. Governance layer
- consent framework
- withdrawal protocol
- community-sensitive data logic
- basic governance notes

### 3. Technical planning layer
- basic data pipeline notes
- initial tech stack
- simple execution environment
- reproducible future direction

## What is not included yet
- full production system
- large-scale speech model training
- mobile app deployment
- public platform launch
- advanced model evaluation

## Success criteria for MVP
- project structure exists and is organized
- base documents are complete
- outreach materials are ready
- governance logic is documented
- technical path is defined at a basic level
